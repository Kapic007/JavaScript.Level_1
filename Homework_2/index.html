Урок 2. Основные операторы JavaScript

  1. Дан код: 
            var a = 1, b = 1, c, d;
        1.1 c = ++a; alert(c); // 2
        1.2 d = b++; alert(d); // 1
        1.3 c = (2+ ++a); alert(c); // 5
        1.4 d = (2+ b++); alert(d); // 4
        1.5 alert(a); // 3
        1.6 alert(b); // 3
     Почему код даёт именно такие результаты? 

              1.1 префиксный инкремент сначала увеливает значение а, а потом присваивает его переменной с
          1.2 сначала присваивается значение b переменной d, а потом постфиксный инкремент увеливает значение b
          1.3 префиксный инкремент сначала увеливает значение а (а==3), потом прибавляет 2, и присваивает переменной с
          1.4 сначала к 2 прибавляется значение b (b==2), потом присваивается переменной d, а потом постфиксный инкремент увеливает значение b
          1.5 с момента обьявления var a = 1, префиксный инкремент два раза увеличил данную переменную
          1.6 с момента обьявления var b = 1, постфиксный инкремент два раза увеличил данную переменную

  2. Чему будет равен x в примере ниже? 
              var a = 2;
          var x = 1 + (a *= 2);

    a *= 2 равнозначно a = а * 2 поэтому x = 1 + (2 * 2) = 5
